Doughmination Windows Setup

This script will debloat, activate, and customize your Windows installation.

Run this command in PowerShell (as Administrator recommended):

irm https://setup.doughmination.win | iex

What does this do?

Step 1: Debloats Windows using Raphi's Windows Debloater

Step 2: Activates Windows using Microsoft Activation Scripts

Step 3: Customizes Windows (Dark Mode, Hidden Files, File Extensions, Taskbar, Wallpaper, Theme Colors)

Step 4: Installs Chocolatey + Node.js v24.13.0 (with npm, yarn, pnpm)

Step 5: Opens Ninite for additional software installation
